- for 문: 조건에 충족(true)하면 실행문을 반복처리
for( 초기값; 조건문; 증감식 ){ 실행문 }
 예시) for(let i = 1; i <= 3; i++){ }
순서도: 1. 초기값 실행 -> 2.조건문 -> 3.실행문 실행 -> 4.증감식
       ->5.조건문 ->6.실행문 실행->7.증감식 ->8.조건문->9.실행문 ->10.증감식 /종료
 - for(){ for( ){ } } 중첩 가능
      상위 for문이 1번 실행될때 마다 하위 for문은 전체 반복
      for( let i = i; i <= 3; i++){
         console.log(i);
         for(let a =1; a <= 2; a++){
            console.log(a);
         }
      }
 - break: 가장 가까운 반복문 탈출/종료
 - continue: 가장 가까운 증감식으로 이동
 - for( ; ; ){ }: 무한루프/반복
