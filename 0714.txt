- 객체란? object : 어떠한 대상의 속성(정보/상태) 정의 하는 것
   ㄴ { } 이용한 자료의 타입
   ㄴ { 속성명 : 속성값 , '속성명' : 속성값 }  -특수문자가 들어갈시  ' ' 문자처리
   ㄴ 객체는 주로 변수/상수에 저장/참조, let / const 변수 = { }
   ㄴ .(접근/도트/참조 연산자) 통해 객체내 속성 값 호출, 변수.속성명
   ㄴ Object.keys( 객체 ) : 모든 속성명 반환,
   ㄴ Object.values( 객체 ) : 모든 속성값 반환
   ㄴ 객체내 속성 추가/수정 : 변수.속성명 = 새로운 값  , delete 변수.속성명
   ㄴ 중첩 : { ket : [ ], key: { }, key : function(){} }, 속성값에는 다양한 타입이 대입된다. 
   ㄴ 속성명 존재 여부 확인 : '속성명' in 객체 -> true / false 반환
